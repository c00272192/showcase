Technical Details

The technology and methodologies behind LiDAR property assessment

Technical Approach

This research project explores the viability of consumer-grade LiDAR and photogrammetry for professional property assessment. The technical implementation involves several key components and processes that enable accurate, accessible documentation of derelict properties.

Hardware Used

  • iPhone 13 Pro - Primary LiDAR scanning device
  • iPad Pro 12.9" (2021) - Secondary scanning device for comparison
  • Leica Disto X4 - Professional laser measuring tool for validation
  • MacBook Pro M1 - For processing and model optimization

Software Stack

  • Polycam - iOS app for LiDAR scanning and initial model creation
  • Scaniverse - Alternative iOS app used for comparative scanning
  • Meshroom - Photogrammetry processing software
  • Blender - 3D model optimization and cleanup
  • Sketchfab - Web platform for hosting and sharing 3D models

Scanning Methodology

  1. Pre-scan preparation - Property cleanup, lighting setup, reference markers
  2. Systematic scanning - Room-by-room approach with 25% overlap between scans
  3. Detail capture - Special attention to architectural features and structural damage
  4. Validation measurements - Taking reference measurements with traditional tools
  5. Post-processing - Model cleanup, alignment, and optimization

Data Processing Pipeline

Raw scan data undergoes several processing stages:

  1. Initial scan registration - Aligning multiple scans into cohesive model
  2. Noise reduction - Removing scanning artifacts and outliers
  3. Mesh optimization - Reducing polygon count while preserving detail
  4. Texture mapping - Applying photographic data to 3D geometry
  5. File conversion - Preparing optimized files for web delivery
  6. Annotation - Adding measurement points and condition notes

Challenges & Solutions

Low Light Conditions

Challenge: Many derelict properties lack electricity, creating dark scanning environments.

Solution: Utilized LiDAR's infrared capabilities supplemented with portable battery-powered LED panels for critical areas.

Reflective Surfaces

Challenge: Windows and mirrored surfaces causing scanning artifacts.

Solution: Temporary application of non-reflective coverings and algorithmic cleaning in post-processing.

Complex Geometries

Challenge: Ornate architectural details failing to scan accurately.

Solution: Supplemental photogrammetry capture and manual model adjustment for complex features.

Data Size Management

Challenge: Raw scans producing extremely large files (5-10GB per property).

Solution: Developed a decimation workflow that preserves critical measurements while reducing file size by up to 85%.

Future Technical Developments

This research has identified several promising directions for future technical improvements:

  • Integration with Building Information Modeling (BIM) platforms for direct import
  • Automated structural damage detection using machine learning algorithms
  • Development of a measurement API for programmatic access to dimensional data
  • Cloud-based processing to reduce hardware requirements
  • Implementation of augmented reality visualization for on-site assessment
